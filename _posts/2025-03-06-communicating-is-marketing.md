管理者的沟通是一个市场行为。

市场行为的目标，是激发行动。

绝大多数时候，我们无法使对方做出自己期望的行动。 <br/>
比如，一家车企的市场活动，与潜在买家买车行动之间，很难有直接联系。

市场行为的重点是激发。

不是激发对自己工作的认同。 <br/>
不是激发对自身处境的同情。 <br/>
不是激发对管理权威的敬畏。 <br/>

而是：
激发对共同处境的认同。 <br/>
激发对共同敌人的愤慨。 <br/>
激发对共同愿景的向往。 <br/>
激发对面临共同困难的雄心。

激发之所以有效，是对这些共同处境、远景、困难的深刻理解和认同。 <br/>
它源于自身工作。 <br/>
它成长于对工作服务对象的深刻的思考。 <br/>
它有其自身、不被判断的价值。

管理者的工作，大部分都在于激发。 <br/>
对于智力密集型工作，工作成果通常难以度量。 <br/>
管理者要将自己工作的重心 <br/>
要初步从以身作则， <br/>
超越到激发人们行动中

在这个意义上，每个管理者都是市场者。 <br/>
不是做销售，做服务，做产品，做技术

而是 传递能量。
